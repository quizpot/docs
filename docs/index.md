# Introduction

Quizpot is an open-source application that allows you to create quizzes without limits, on your own terms.

## Requirements

- 2 CPU Cores
- ~2GB of RAM
- 2GB of Storage

> [Detailed requirements](/requirements)

## Quick Start

Easily run Quizpot with the following commands:

> Note: This assumes you have dependencies installed on your machine. [If not, follow the instructions here](/installation)

```sh
# Clone the repository
git clone https://github.com/quizpot/quizpot.git

# Go into the repository
cd quizpot

# Copy the .env.example file to .env
cp .env.example .env

# Install & run the application in development mode
npm install && npm run dev
```
