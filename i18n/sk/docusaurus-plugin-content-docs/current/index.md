# Úvod

Quizpot je aplikácia, ktorá umožňuje vytvárať kvízy bez obmedzení, na vlastné podmienky.

## Požiadavky

* 2 CPU Jadrá
* ~2 GB RAM
* 2 GB Úložisko

> [Detailné požiadavky](/requirements)

## Rýchly Štart

Spustite Quizpot jednoducho pomocou nasledujúcich príkazov:

> Poznámka: Predpokladá sa, že máte na svojom zariadení nainštalované závislosti. [Ak nie, postupujte podľa pokynov tu](/installation)

```sh
# Naklonujte repozitár
git clone https://github.com/quizpot/quizpot.git

# Prejdite do repozitára
cd quizpot

# Skopírujte súbor .env.example do .env
cp .env.example .env

# Nainštalujte a spustite aplikáciu vo vývojovom režime
npm install && npm run dev
```
